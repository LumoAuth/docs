# API Specification

Interactive OpenAPI 3.1 documentation for the LumoAuth Tenant Admin API.
Browse all 100+ endpoints, view request/response schemas, and try examples.

:::tip[How to Use This Specification]
- Use the left sidebar to navigate between resource types (Users, Roles, Clients, etc.)
- Click on any endpoint to see detailed request/response examples
- Review the schemas section for complete data models
- All endpoints require Bearer token authentication
:::


The OpenAPI specification for the LumoAuth Tenant Admin API is available for download:

```
GET /docs/admin/openapi.yaml
```

You can use this specification with tools like [Swagger UI](https://swagger.io/tools/swagger-ui/),
[Redoc](https://redocly.com/redoc), or import it into [Postman](https://www.postman.com/)
for interactive API exploration.
